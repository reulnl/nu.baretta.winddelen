De Windcentrale winddelen

Laat Homey de opbrengst van je Windcentrale 'winddelen' (aandelen in een windmolen, zie https://www.windcentrale.nl/) monitoren. Met deze app kun je triggers maken op basis van het huidige vermogen en, misschien nog handiger, productiestatistieken bekijken met Insights.

Gebaseerd op de app 'Zonnepanelen' van Diederik Bakker.


Een windmolen toevoegen

Selecteer de molen waaraan je deelneemt en vul het aantal 'winddelen' in.


Functies

 - Huidige opbrengst
   Homey toont de output van je winddelen als sensor. Deze app bevat momenteel de volgende metrics (allemaal per 'winddeel'): huidig vermogen, huidige energieproductie (dagelijks reset), de jaarproductie en de totale productie.

 - Stromen
   De Winddelen app kan gebruikt worden als sensor binnen Homey. Hierdoor kun je logica gebruiken gebaseerd op je huidige opbrengst.

 - Inzichten
   Homey haalt elke minuut gegevens op vanaf de 'Windcentrale', maar logt alleen nieuwe gegevens. Je kan meer dan één windmolen toevoegen, en elke molen wordt in een aparte grafiek vastgelegd.



ENGLISH

De Windcentrale winddelen

Let Homey monitor the output of your Windcentrale 'winddelen' (shares in a windmill, see https://www.windcentrale.nl/). This app allows you to create triggers based on the current power output, and, perhaps more useful, view production statistics using Insights. 

Based on the 'Solar panels' app by Diederik Bakker.


Adding a windmill

Select the windmill you participate in and enter the number of 'winddelen' (shares).


Features

 - Current production data
   Homey shows your winddelen output as a sensor. This app currently includes the following metrics (all per 'winddeel' or share): current power, current energy production (reset daily), the yearly production, and the lifetime production. 

 - Flow
   The Winddelen app can be used as a sensor for Homey. This allows you to use logic on your production data. 

 - Insights
   Homey retrieves data from the 'Windcentrale' every minute, but only logs new data. You can add more than one windmill, and each will be logged in a separate graph.
